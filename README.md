# html-test-host

This branch hosts the most recent HTML report generated by
[codebase-teacher](https://github.com/mpallone/codebase-teacher) for
GitHub Pages.

- Only `index.html` at the branch root is served. Each
  `/teach-evaluate-push {path} html` run overwrites it.
- `.nojekyll` disables GitHub Pages' Jekyll processing so files
  beginning with `_` are served as-is.
- Do **not** land source-code changes here. For those, use `main`.

To enable hosting, configure the repo's GitHub Pages settings to serve
from branch `html-test-host` at path `/ (root)`.
